基于手势交互的多媒体梦境修复艺术项目
梦境
木马
《寻找失落的梦境》是一份以「梦境修复」为叙事核心的多媒体交互艺术作品。作品融合了手势识别技术与拼图叙事，构建了一个由视觉、听觉、心觉三层感知交织而成的梦境世界。

🎠 第一章 · 停滞的乐园

九块散落的拼图碎片，需通过鼠标拖拽或手势抓取，将旋转木马拼合完整。找回被遗忘的童年快乐。

🎻 第二章 · 静默的音符

七根琴弦悬浮于画面，手指拂过即奏响安眠曲。金色光点引导弹奏顺序，十四音符逐一亮起，唤醒沉睡的旋律。

🏮 第三章 · 孔明之灯

双手合十许下心愿，六盏孔明灯从夜色中升起。每点亮一盏，便触发一段纯真愿望，温暖的光照亮记忆之门。

创作工具：HTML / CSS / JavaScript / MediaPipe Hands / Web Audio API / Canvas
交互方式：手势识别 · 鼠标拖拽 · 触屏操作 · 摄像头实时捕捉
设计风格：暖金梦境 · 星空粒子 · 水墨质感 · 行书字体
